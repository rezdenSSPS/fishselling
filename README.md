## 🎨 Style Guide

- **Figma:** *(add your Figma link here)*
- **Colors:** Teal primary (#006a65), Ocean gradient (#006a65 → #00b5ad), Coral accent (#fe7e4f), Neutral surfaces (#f4fbf9)
- **Typography:** Plus Jakarta Sans (300–800)
- **Spacing:** 4px base unit (Tailwind default scale)
- **Components:** Rounded cards, pill buttons, glass navigation, toast notifications

## ✨ Features

- **Product gallery** with bento grid layout
- **Category filter** (All / Fish / Shellfish) — JS interactive
- **Shopping cart** sidebar with add/remove and total calculation
- **Newsletter form** with email validation
- **Responsive design** — mobile & desktop
- **Accessible** — WCAG AA (skip links, ARIA labels, focus management, semantic HTML)
- **Smooth animations** — scroll-triggered card entrances, micro-interactions

## 🛠️ Tech Stack

- **HTML5** — semantic markup
- **Tailwind CSS** — via CDN (v3)
- **JavaScript** — vanilla, in separate `script.js`
- **Google Fonts** — Plus Jakarta Sans
- **Material Symbols** — icon set
